# Darto Car Dealership

Este é um projeto simples de uma API de concessionária de carros construída com o framework Darto.

## Como clonar

Para clonar o repositório para a sua máquina local, execute o seguinte comando no terminal:

```bash
git clone https://github.com/carlosxfelipe/darto-car-dealership.git
```

## Como instalar os pacotes

Acesse o diretório do projeto clonado e instale as dependências necessárias utilizando o gerenciador de pacotes do Dart:

```bash
cd darto-car-dealership
dart pub get
```

## Como rodar o servidor

Para iniciar a aplicação, execute o arquivo principal do servidor com o comando abaixo:

```bash
dart run bin/server.dart
```

O servidor será iniciado na porta 3000 (`http://localhost:3000`).

## Documentação (Swagger)

A documentação interativa da API pode ser acessada através da rota `/docs`.

Após iniciar o servidor, abra o seu navegador e acesse:

```text
http://localhost:3000/docs
```
